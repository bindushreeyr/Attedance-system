# Smart Attendance System

A production-ready Django web application for managing student attendance.

## Features
- Student management
- Daily attendance tracking
- Attendance reports with percentage
- PDF report generation
- PostgreSQL-backed persistent storage
- Deployed on Render

## Tech Stack
- Django
- PostgreSQL
- HTML, Bootstrap
- ReportLab
- Gunicorn
- Render
